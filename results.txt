=== BEFORE/AFTER COMPARISON DATA ===

WITHOUT HONEYPOT:
- Attempts before alert: 3
- Time to first alert: ~15 seconds
- Detection method: auth.log only
- Data captured: IP address only
- Command logging: No
- Credentials captured: No

WITH HONEYPOT:
- Attempts before alert: 3
- Time to first alert: ~3 seconds
- Detection method: Cowrie docker logs (real time)
- Data captured: IP + username + password + commands
- Command logging: Yes
- Credentials captured: Yes (123456, password)

KEY FINDINGS:
- Honeypot detection is 5x faster
- Honeypot captures attacker credentials
- Honeypot logs attacker commands
- Honeypot alerts are always real (no false positives)
- Combined system detects both real SSH and honeypot attacks
